## 🐛 常见问题

### 1. 端口被占用
**问题**: 3000 或 8081 端口已被占用

**解决**:
```cmd
# 查找占用进程
netstat -ano | findstr :3000
netstat -ano | findstr :8081

# 结束进程
taskkill /PID <进程ID> /F
```

### 2. 数据库连接失败
**问题**: 无法连接到 MySQL

**解决**:
- 检查 MySQL 服务是否启动
- 确认数据库密码为 `123456`
- 确认数据库 `mychat` 已创建

### 3. 登录失败
**问题**: 输入正确账号密码后仍无法登录

**解决**:
- 检查后端服务是否运行
- 打开浏览器控制台查看错误
- 清除浏览器缓存重试

### 4. 图表不显示
**问题**: 数据看板图表空白

**解决**:
- 确认 echarts 已安装
- 检查 API 数据返回格式
- 查看浏览器控制台错误
